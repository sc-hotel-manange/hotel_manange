# 汤神酒店辅助订购系统

基于供应商提供的数据和接口完成酒店辅助订购系统的开发。

## 上手指南

以下指南将帮助你在本地机器上安装和运行该项目，进行开发和测试

## 安装要求

* JDK1.8
* Maven
* MySQL 5

## 运行项目

1. 运行`HotelManagerApplication.java`
2. 在浏览器中访问 localhost:8080/user/

## 使用到的框架

* Springboot - Web框架
* Maven - 依赖属性管理
* Mybatis - 持久层框架
